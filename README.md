## Best Practices Learned
- Use **12‚Äì16+ characters**
- Use mix of:
  - Uppercase (A‚ÄìZ)
  - Lowercase (a‚Äìz)
  - Numbers (0‚Äì9)
  - Symbols (!@#$%)
- Avoid common words
- Never use personal info (DOB, name, mobile)
- Use **Passphrases** like: `Sunset$River@2025Flows`
- Use **Password Manager**
- Enable **Multi-Factor Authentication (MFA)**

## Common Password Attacks Researched

- Brute force attack

- Phishing attack
- Password reuse attacks

## Conclusion
By testing multiple passwords, I learned how complexity and length improve security.
